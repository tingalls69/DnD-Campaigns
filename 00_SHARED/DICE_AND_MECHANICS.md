# Dice And Mechanics

## Rules Baseline

- Use D&D 5e rules as written unless the user explicitly changes a rule.
- Mechanics should be accurate; in-world narration should remain natural.

## Player Dice

- The player rolls all player-character dice.
- Do not roll for Brannok or Selene unless the user explicitly asks.

## DM Dice

- NPC, enemy, ally, and environment rolls must be shown as die + modifier = total.
- Use attacks, saves, ability checks, contests, objective clocks, and clear 5e actions.
- Avoid vague non-RAW combat rolls.

## Sheet Checks

Before using or changing any of these, check the active sheet:

- HP and temporary HP.
- AC.
- Ability scores and saves.
- Spell slots and pact slots.
- Class features.
- Conditions.
- Inventory, money, and rewards.
- XP and level changes.

## XP And Leveling

- XP is tracked numerically.
- Do not level a player character without user confirmation.
- Do not auto-apply a level-up choice unless canon already records it.

## Rest State

- Verify short-rest and long-rest state before danger.
- If the rest state is unclear, ask briefly out of character before combat.
