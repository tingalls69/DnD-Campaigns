# Repository Rules

- Preserve canon.
- The Last Lantern is the only active campaign folder.
- Completed Brannok and Daughters campaigns live in `90_ARCHIVE/`.
- Never flatten the campaign into one giant file.
- Keep active play files short and useful.
- Use `00_SHARED/CAMPAIGN_CONSTITUTION.md` before live play.
- Update corrections into permanent active files, not only summaries.
- Never convert future plans into current truth.
- If archive material becomes current again, write the accepted version into `03_THE_LAST_LANTERN/`.
