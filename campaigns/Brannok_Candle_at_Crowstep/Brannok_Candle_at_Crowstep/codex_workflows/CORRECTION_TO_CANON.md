# Correction To Canon

When the user corrects the DM, do not treat it as temporary chat context.

## Classify The Correction

- Character voice
- Relationship
- Mechanics
- Plot fact
- NPC knowledge
- Inventory/resource
- Retcon

## Update The Correct Permanent File

- Protected character file
- `03_CURRENT_STATE.md`
- `10_CANON_LOCKED_FACTS.md`
- `06_PLAYER_CHARACTER_SHEET.md`
- `09_RELATIONSHIP_MAP.md`
- `08_ACTIVE_PLOT_THREADS.md`
- `02_SESSION_PACKET.md`

## Required Note

Add a short note explaining what changed and why. If the correction supersedes older material, preserve the older version in source notes or `archives/` rather than deleting it outright.
