# Session Update Checklist

After every session:

1. Read the latest transcript/session notes.
2. Update `02_SESSION_PACKET.md` first.
3. Update `04_CURRENT_CAST.md`.
4. Update `03_CURRENT_STATE.md`.
5. Append to `14_RUNNING_SESSION_LOG.md`.
6. Update character sheets for HP, XP, resources, inventory, money, spells, conditions, rests, and level changes.
7. Update protected character files if any recurring NPC changed or was corrected.
8. Update NPC knowledge boundaries.
9. Update relationship map.
10. Update active plot threads.
11. Move resolved or obsolete material to `archives/`.
12. Add canon locked facts only when permanent.
13. Produce a short summary of changed files.

Do not leave corrections trapped inside the session summary. Put them where future live play will actually load them.
