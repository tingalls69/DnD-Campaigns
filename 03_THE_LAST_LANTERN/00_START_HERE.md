# The Last Lantern — Start Here

This folder is the active source of truth for the combined Brannok Ashhelm / Selene Vaer campaign.

## Read Before Running Any Session

Read these files in order:

1. `DM_OPERATING_RULES.md`
2. `CURRENT_STATE.md`
3. `PROTAGONISTS.md`
4. `CHARACTER_SHEETS.md`
5. `COMPANIONS.md`

7. `FACTIONS.md`
8. `STORY_ARC.md`
9. Relevant archived source files only when an active file says a detail needs verification.

## Core Running Principle

Player agency is the highest priority. Never decide Brannok or Selene's actions, dialogue, thoughts, feelings, or choices.

Start from the current scene. Do not open with a lore dump. Present an immediate concrete situation with people at stake.

## Current Campaign Shape

The Last Lantern is a character-driven road-rescue fantasy about people using official roads, records, and custody systems to make vulnerable people disappear, and about the party becoming the light that refuses to leave those people behind.

## Fresh AI Reminder

If running from a fresh chat, load this folder first and obey `DM_OPERATING_RULES.md` before generating narration, combat, NPC dialogue, or story planning.
